# /discord-roles Command

List all roles in the Discord server.

## Usage

```
/discord-roles
```

## Behavior

When this command is invoked:

1. Use Discord MCP tools:
   - List all server roles

2. Present roles in hierarchy order:
   - Role name
   - Color
   - Member count
   - Key permissions
   - Mentionable status

3. Offer to:
   - Create a new role
   - Edit an existing role
   - Assign a role to a user
   - Delete a role

## Examples

```
/discord-roles
```

## Output

Provide:
1. List of roles in hierarchy order
2. Permission summary for each role
3. Member counts
4. Options for role management
